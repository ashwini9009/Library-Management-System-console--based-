# Library Management System (Java - Console Based)

##  Project Description

This is a simple console-based Library Management System developed in Java.
It allows librarians to manage books, members, and borrowing operations using basic Object-Oriented Programming concepts and file handling.

---

##  Features

* Add and remove books
* Register library members
* Borrow and return books
* Due date tracking (7 days)
* Overdue fine calculation (₹5 per day)
* Search books by title
* View library statistics
* Data stored using file handling (persistent storage)

---

## Technologies Used

* Java
* OOP Concepts (Encapsulation, Classes, Objects)
* File I/O (BufferedReader, BufferedWriter)
* Collections (ArrayList, HashMap)

---

##  Project Structure

* `Book.java` → Book details
* `Member.java` → Member details and borrowed books
* `Library.java` → Core logic and file handling
* `Main.java` → User interface (console menu)

---

##  How to Run

1. Open project in any Java IDE (Eclipse / IntelliJ / VS Code)
2. Compile all `.java` files
3. Run `Main.java`
4. Use the menu options in console

---

##  Data Storage

* `books.txt` → Stores book data
* `members.txt` → Stores member data

---

##  Example Operations

* Add book → Enter ISBN, title, author
* Borrow book → Enter member ID and ISBN
* Return book → System calculates fine if late

---

##  Learning Outcomes

* Applied Object-Oriented Programming
* Implemented file handling for data persistence
* Built a real-world mini project using Java
